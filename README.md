## Installation

Grab your firebase and TMDB credentials and create a .env file in the root and add them. While naming these make sure to append 'REACT_APP' at the start.

```
npm install
```

##### Run on http://localhost:3000

```
npm start
```
#### Build for prod
```
npm build
```
